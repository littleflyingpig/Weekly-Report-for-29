## 一、本周学习概览

本周主要跟随 Java SE 课程推进，从自己尚未掌握的知识点切入，通过网课系统学习，具体内容如下：

- **集合框架**：学习了 ArrayList、LinkedList、HashMap 的底层结构，重点研究了 HashMap 的扩容机制与哈希冲突解决方案。

- **Stream 流**：掌握了 filter、map、collect 等常用操作，能够用流式处理替代传统的 for 循环，代码更简洁易读。

- **IO 流**：理清了字节流与字符流的区别，学会了使用 BufferedReader 高效读取文件，以及 Properties 配置文件的基本用法。

- **日志技术**：配置了 Logback，熟悉了日志级别（TRACE / DEBUG / INFO / WARN / ERROR），逐步摒弃 System.out.println 的调试方式。

- **多线程**：学习了线程创建的三种方式，理解了 synchronized 与 Lock 的区别，以及线程池核心参数的含义。

## 二、遇到的问题与解决

**问题**：IO 类繁多，容易混淆，不清楚什么场景该用哪个类。

**解决**：根据类名后缀进行分类记忆（如：以 Input 结尾 / 带 Reader → 读，以 Output 结尾 / 带 Writer → 写），并结合实际练习加深理解。
